

### LIVE LINK

https://ecl-clone.vercel.app/

## Tech Stack

React, TailwindCSS, Framer motion, GSAP

## Run Locally

Clone the project

```bash
  git clone https://github.com/segunlawal/ecl-clone.git
```

Go to the project directory

```bash
  cd ecl-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
